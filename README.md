# Flask 项目端口监控与启动工具

本项目用于在本地通过网页监控多个项目端口状态，并可一键执行 bat 脚本启动项目。

## 功能简介
- 实时检测多个项目端口是否开放
- 支持点击按钮执行对应 bat 脚本启动项目
- 启动后页面顶部弹出提示，显示启动用时
- 端口开放时可直接点击访问项目

## 目录结构
```
batstart/
├── app.py                # Flask 主程序
├── project_config.py     # 项目端口与启动脚本配置
├── start_a.bat           # 示例项目A启动脚本
├── start_b.bat           # 示例项目B启动脚本
├── static/
│   ├── main.js           # 前端交互脚本
│   └── style.css         # 页面样式
└── templates/
    └── index.html        # 主页面模板
```

## 使用方法
1. 安装依赖：
   ```bash
   pip install flask
   ```
2. 启动服务：
   ```bash
   python app.py
   ```
3. 浏览器访问：
   [http://127.0.0.1:9999](http://127.0.0.1:9999)

## 配置说明
- 在 `project_config.py` 中配置每个项目的名称、端口和启动脚本路径。
- 各项目的 bat 脚本需放在同目录下，内容可自定义。

## 注意事项
- 启动脚本需保证有执行权限。
- 端口检测仅检测本机 127.0.0.1。
- bat 脚本执行为异步，不会阻塞页面。
